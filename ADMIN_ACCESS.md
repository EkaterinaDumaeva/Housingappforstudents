## How to Manage Admin Accounts (Master Admin Only)

### Creating a New Admin Account:

1. Log in with master admin credentials
2. Click **"Manage Admins"** button in the top-right corner of the dashboard
3. Click **"Create New Admin"** button
4. Fill in the required information:
   - Full Name (e.g., "John Smith")
   - Email Address (e.g., "john@example.com")
   - Password (minimum 8 characters)
5. Click **"Create Admin Account"**
6. The new admin can now log in with their credentials

### Deleting an Admin Account:

1. Go to **"Manage Admins"** section
2. Find the admin account you want to remove
3. Click the **"Delete"** button next to their account
4. Confirm the deletion
5. The admin will immediately lose access

### Viewing Admin Passwords:

- Click the **eye icon** next to a regular admin's password to reveal it
- Click again to hide the password
- **Note:** Master admin password can NEVER be viewed

### Important Notes:

⚠️ **Admin Account Best Practices:**
- Only create admin accounts for trusted team members
- Use strong, unique passwords for each admin account
- Regularly review the list of active administrators
- Remove admin access immediately when team members leave
- Never share the master admin credentials

⚠️ **Security Reminders:**
- All admin actions are logged
- Admins have full access to user data and platform controls
- Exercise caution when granting admin privileges
- The master admin account should be used by the platform owner only
