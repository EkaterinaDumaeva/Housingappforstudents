## ⚡ Ultra-Quick Start

**Already know what you're doing?**

```bash
# 1. Create Supabase project → get credentials
# 2. Upload to Replit
# 3. In Replit shell:

rm __figma__entrypoint__.ts
mv vite.config.replit.ts vite.config.ts
mv utils/supabase/info.replit.tsx utils/supabase/info.tsx

# 4. Set 4 secrets in Replit Secrets tab
# 5. Run:

npm install
npm run dev
```

Done! ✅
